퍼블리싱 가이드

들여쓰기/빈줄

들여쓰기 : 1탭 들여쓰기 하며, 1탭의 크기는 공백 4칸으로 설정한다.

빈줄 : 가독성을 고려하여 빈 줄을 추가 가능, 단 연속적으로 1줄을 초과하지 않는다.

단위

font 단위는 ‘em’ 사용으로 원칙한다.

    body {font-size:16px;} /* 16px 기준 */
    div {font-size:1em;} /* body에서 상속된 기준값으로 div의 1em은 16px */
    div.section {font-size:0.75em;} /* body에서 상속된 기준값으로 div의 0.75em은 12px */
    div.section p {font-size:1em;} /* 상위 엘리먼트인 div.section의 기준값을 상속받아 div.section 내의 p의 1em은 12px */

LABEL 태그사용

그림을 예시로 title과 input태그는 <label> 태그를 이용하는 것을 원칙한다.

    <label for="idName">닉네임
    <input type="text" id="idName">
                            
    <label for="passwordNum">비밀번호
    <input type="password" id="passwordNum">

    <iframe src="" title=""></iframe>
    <img src="img.png" alt="대체텍스트">
    <input type="text" name="" id="" value="" title="" class="" disabled="">

CSS 들여쓰기/줄 바꿈

CSS 작성시 들여쓰기는 허용하지 않는다.

공통으로 사용되는 속성의 선택자간의 줄 바꿈은 허용하며 이 외의 줄바꿈은 허용하지 않는다.

    .class-name,
    .class-name .child-class {font-size:1.25em; letter-spacing:-1px; margin:0.5em 0 0 0;}
                                
    .parents-class .child-class {text-align:center;}

CSS 공백의 사용과 세미콜론(;)

선택자 간, 중괄호간 한칸의 공백을 사용, 중괄호 안쪽 좌우에는 공백은 사용 X

마지막 선언된 속성에도 세미콜론(;) 을 사용

    .class-name {color:#eee; font-size:1.2em;}

    /*잘못된 예시*/
    .class-name {   color:#eee;font-size:1.2em;  }

CSS 주석
